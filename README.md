## Unary operator:
```
•	Unary Plus Operator (+):”~+”

•	Unary Minus Operator (-):”~-”

•	Logical Complement (negation) Operator (!):”-!”

•	Bitwise Complement (negation) Operator (~):”~~”

•	Prefix Increment (++) and Decrement (--) Operator:"~++" and "~--"
```
